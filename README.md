# 硕士毕业论文latex目录模板

本latext目录，可用作统计类、数学类、经济学类硕士、本科毕业论文的参考。

可参考本目录构建自己的章节、图片、参考文献（建议使用`bibtex`）、封面等。

还包含slides的模版，可供参考。

## 编译

使用`xelatex`命令行工具编译`main.tex`文件或者使用TextStudio用户界面进行编译。
